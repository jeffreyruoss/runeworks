## Sprites

Generated by AI (Google Gemini). Use the `/sprite` slash command to add new ones. Gemini sometimes adds fake transparency (checkerboard) — ask Claude Code to run the fix-transparency tool if sprites look wrong.

## Audio

8-bit sound effects are generated in code (no audio files). Design new sounds at https://sfxr.me, then ask Claude Code to add the sound params to `src/audio.ts`.

## INFO

Gold - mine it - the main currency. Used for building and upgrading buildings, and for buying items.
Mana - produce it - used for magic-related buildings and items. Like electricity in Factorio, but more fantasy-themed.
Flow - get it by building fast - fills up the flow meter and when that is full you can upgrade.

Research - produced - for unlocking new buildings and items.
Upgrades - earned from filling flow meter. like skill points: for improving the things you already have, not unlocking new things.
